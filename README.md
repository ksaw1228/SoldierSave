# 장병내일적금 수령액 계산기 :money_with_wings:

[장병내일적금 수령액 계산기](https://ksaw1228.github.io/SoldierSave/)는 군 복무 기간 동안 월 지급 금액을 간편하게 계산할 수 있는 온라인 계산기입니다.

## :star2: 주요 기능

- 총 납입 기간 계산 필요 없이 첫 납입일만 입력하면 간편하게 계산 완료!
- 복무 기간에 따른 수령액과 예상 이자 계산
- 모바일 및 데스크톱 모두 사용 가능한 반응형 디자인

## :arrow_forward: 사용 방법

1. [장병내일적금 수령액 계산기](https://ksaw1228.github.io/SoldierSave/)로 이동합니다.
2. 웹 페이지에서 첫 적금 납일일자와 전역 일자를 선택하세요.
3. 월 납입 금액을 입력하세요.
4. "계산하기" 버튼을 클릭하면 총 수령액이 표시됩니다.

장병내일적금 수령액 계산기로 손쉽게 적금 총 수령액을 확인하세요! :sparkles:
